# Dashboards

Standalone frontend templates you can host on Vercel or Netlify, pointed at your Supabase backend.

This folder is the reusable template and community contribution surface. The maintained unified product app lives in `apps/brain/`.

## Current Dashboards

- `command-center/` - action and planning dashboard
- `data-browser/` - read-only browser for Open Brain tables
- `home-maintenance/` - home maintenance frontend
- `thought-explorer/` - exploratory thought visualization dashboard
- `_template/` - starting point for new dashboard contributions

## Contributing

Dashboards are open for community contributions. See [CONTRIBUTING.md](../CONTRIBUTING.md) for details.

If a dashboard concept graduates into the canonical product app, move or rebuild it in `apps/brain/` deliberately and document that promotion.
